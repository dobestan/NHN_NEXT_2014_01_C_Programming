print\-\_\-gray\-\_\-code(int $\ast$ gray\-\_\-code, 0, B\-I\-T\-\_\-\-S\-I\-Z\-E, 0)으로 시작한다. bit refers to result of X\-O\-R(bit, gray\-\_\-code\mbox{[}start\-\_\-bit\mbox{]})

시작하기에 앞서 십진수(이진수 -\/$>$ 그레이코드)로 표시하기로 한다.\par
코드를 시작하기에 앞서 몇 가지 예시를 통해서 쉽게 구하는 방법을 살펴보자.

\paragraph*{3(11 -\/$>$ 10)을 통해서 재귀적으로 7(111 -\/$>$ ?)을 구하는 과정}

11 -\/$>$ 111, 즉 마지막 자리에 1이 생겼다.\par
X\-O\-R(1 , 1) = 0 이므로 10에다가 0만 붙여주면 된다.\par
7(111 -\/$>$ 100)

\paragraph*{3(11 -\/$>$ 10)을 통해서 재귀적으로 6(110 -\/$>$ ?)을 구하는 과정}

11 -\/$>$ 110, 즉 마지막 자리에 0이 생겼다.\par
X\-O\-R(1, 0) = 1이므로 10에다가 1만 붙여주면 된다.\par
6(110 -\/$>$ 101)

start\-\_\-bit == end\-\_\-bit 인 경우, 우리가 원하는 graycode를 얻었으므로 출력하면 된다.

start\-\_\-bit != end\-\_\-bit 인 경우, 계속 진행한다.

bit는 이진수의 bit 값을 의미한다.\par
출력 순서는 bit 값을 통해서 조정하자.\par
\par
 만약에 end\-\_\-bit == 2 경우, bit 값에 따른 출력 순서는 다음과 같다.\par
0 0 0\par
0 0 1\par
0 1 0\par
0 1 1\par


초기값 설정 0(0 -\/$>$ 0) X\-O\-R(0, 0) = 0

초기값 설정 1(1 -\/$>$ 1) X\-O\-R(0, 1) = 1

X\-O\-R(bit, next\-\_\-bit) = gray\-\_\-code\mbox{[}start\-\_\-bit\mbox{]} 를 구할 수 있다.

X\-O\-R(1, 0) = 1

X\-O\-R(1, 1) = 0
