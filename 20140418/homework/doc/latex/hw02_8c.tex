\subsection{Detailed Description}
Call by Value / Call by Reference Example. 변수 num에 저장된 값의 제곱을 계산하는 함수를 정의하고, 이를 호출하는 main함수를 작성해보자. 단, 여기서는 다음 두 가지 형태로 함수를 정의해야 한다. \par
\par
call-\/by-\/value기반의 Square\-By\-Value\par
Call-\/by-\/reference기반의 Square\-By\-Referce\par
\par
Square\-By\-Value 함수는 인자로 젂달된 값의 제곱을 반홖해야 하면, Square\-By\-Reference 함수는 정수가 저장되어 있는 변수의 주소값을 인자로 받아서 해당 변수에 저장된 값의 제곱을 그 변수에 다시 저장한다.
