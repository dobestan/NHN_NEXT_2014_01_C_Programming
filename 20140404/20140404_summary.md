## 함수의 구조
```c
int main(void) {
  // ** 함수의 내용 **
}
```
** function signature ** 는 크게 리턴값, 인자, 함수 이름으로 구성된다.
- int : return type
- void : parameter
- main : name of function

```c
int Add(int num1, int num2) {
  int result = num1 + num2;
  return result;
}
```
- 일반적으로 C언어에서 함수의 첫 문자는 upper case로 쓴다.
